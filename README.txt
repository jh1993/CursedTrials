To install this mod, click on the green "Code" button on this page, then "Download ZIP". Please rename the "PyrotechnicianTrial-main" folder to "PyrotechnicianTrial" before putting it into your mods folder.

Adds a new archmage trial, Pyrotechnician, where the only available spell is a fire sorcery spell that deals 0 damage in a 0 tile burst.
